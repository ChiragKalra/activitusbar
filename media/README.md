# Logo Icon Sources

The generated `media/generated/activitus-logos.woff` file is built from the SVGs in `media/logo-icons-src`.

- `activitus-gemini.svg`: Wikimedia Commons `Google-gemini-icon.svg`, CC0 1.0.
- `activitus-claude.svg`: copied from the installed Claude Code extension resources.
- `activitus-codex.svg`: copied from the installed OpenAI/Codex extension resources.

Rebuild with:

```sh
npm run build:icons
```
